# Code Explanation

Explique le code demandé en 3 niveaux :

1. **Vue d'ensemble** : Une phrase sur ce que fait le code.
2. **Détail** : Comment ça marche, étape par étape.
3. **Contexte** : Pourquoi cette approche a été choisie.

Utilise des analogies simples si le concept est complexe.
Termine par une question ouverte pour vérifier la compréhension.
