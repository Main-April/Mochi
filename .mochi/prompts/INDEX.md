# Prompts Disponibles

Tu peux charger n'importe lequel de ces prompts en lisant le fichier correspondant.

| Fichier | Description |
|---------|-------------|
| `DESINGS_SKILLS.md` | Guidance design frontend pour UI distinctives |
| `CODE_REVIEW.md` | Revue de code approfondie |
| `DEBUG_HELP.md` | Diagnostic et correction de bugs |
| `REFACTOR.md` | Suggestions de refactoring |
| `EXPLAIN_CODE.md` | Explication claire de code |

**Comment les utiliser :**
- Si tu veux plus de détails sur un sujet, charge le fichier correspondant avec l'outil `read`.
- Tu peux aussi me demander de te guider sur un de ces thèmes sans charger le fichier.
