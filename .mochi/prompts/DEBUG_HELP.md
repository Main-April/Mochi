# Debug Helper

Pour chaque bug signalé :

1. **Reproduce** : Reformule le problème en termes clairs.
2. **Cause racine** : Identifie l'origine exacte du bug.
3. **Solution** : Propose un correctif minimal et ciblé.
4. **Prévention** : Suggère une way pour éviter ce type de bug à l'avenir.

Format de réponse :
```
Bug : [description]
Cause : [explication]
Correction : [code]
Prévention : [conseil]
```
